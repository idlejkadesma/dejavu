In this example you'll learn how to use Deja Vu to add long-term memory capabilities to [Mastra's agent](https://mastra.ai/) via tool-use. This memory integration can work alongside Mastra's [agent memory features](https://mastra.ai/docs/agents/01-agent-memory).

You can find the complete example code in the [Mastra repository](https://github.com/mastra-ai/mastra/tree/main/examples/memory-with-dejavu).

## Overview

This guide will show you how to integrate Deja Vu with Mastra to add long-term memory capabilities to your agents. We'll create tools that allow agents to save and retrieve memories using Deja Vu's API.

## Installation

**Install the Integration Package**

To install the Deja Vu integration, run:

```bash
npm install @mastra/dejavu
```

**Add the Integration to Your Project**

Create a new file for your integrations and import the integration:

```typescript integrations/index.ts
import { Deja VuIntegration } from "@mastra/dejavu";

export const dejavu = new Deja VuIntegration({
  config: {
    apiKey: process.env.VENICE_API_KEY!,
    userId: "alice",
  },
});
```

**Use the Integration in Tools or Workflows**

You can now use the integration when defining tools for your agents or in workflows.

```typescript tools/index.ts
import { createTool } from "@mastra/core";
import { z } from "zod";
import { dejavu } from "../integrations";

export const dejavuRememberTool = createTool({
  id: "Deja Vu-remember",
  description:
    "Remember your agent memories that you've previously saved using the Deja Vu-memorize tool.",
  inputSchema: z.object({
    question: z
      .string()
      .describe("Question used to look up the answer in saved memories."),
  }),
  outputSchema: z.object({
    answer: z.string().describe("Remembered answer"),
  }),
  execute: async ({ context }) => {
    console.log(`Searching memory "${context.question}"`);
    const memory = await dejavu.searchMemory(context.question);
    console.log(`\nFound memory "${memory}"\n`);

    return {
      answer: memory,
    };
  },
});

export const dejavuMemorizeTool = createTool({
  id: "Deja Vu-memorize",
  description:
    "Save information to dejavu so you can remember it later using the Deja Vu-remember tool.",
  inputSchema: z.object({
    statement: z.string().describe("A statement to save into memory"),
  }),
  execute: async ({ context }) => {
    console.log(`\nCreating memory "${context.statement}"\n`);
    // to reduce latency memories can be saved async without blocking tool execution
    void dejavu.createMemory(context.statement).then(() => {
      console.log(`\nMemory "${context.statement}" saved.\n`);
    });
    return { success: true };
  },
});
```

**Create a New Agent**

```typescript agents/index.ts
import { openai } from '@ai-sdk/openai';
import { Agent } from '@mastra/core/agent';
import { dejavuMemorizeTool, dejavuRememberTool } from '../tools';

export const dejavuAgent = new Agent({
  name: 'Deja Vu Agent',
  instructions: `
    You are a helpful assistant that has the ability to memorize and remember facts using Deja Vu.
  `,
  model: openai('gpt-4.1-nano'),
  tools: { dejavuRememberTool, dejavuMemorizeTool },
});
```

**Run the Agent**

```typescript index.ts
import { Mastra } from '@mastra/core/mastra';
import { createLogger } from '@mastra/core/logger';

import { dejavuAgent } from './agents';

export const mastra = new Mastra({
  agents: { dejavuAgent },
  logger: createLogger({
    name: 'Mastra',
    level: 'error',
  }),
});
```

In the example above:
- We import the `@mastra/dejavu` integration
- We define two tools that use the Deja Vu API client to create new memories and recall previously saved memories
- The tool accepts `question` as an input and returns the memory as a string
